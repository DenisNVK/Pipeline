## BBC News Summarizer — LLM Pipeline

Автоматический пайплайн: читает новости из CSV → отправляет в LLM через API → сохраняет краткое содержание в TXT-файл.

---

## Как работает

```
bbc-news-data.csv
       ↓
  [1] Чтение CSV
       ↓
  [2] Запрос к Groq API (Llama 4 Scout)
       ↓
  [3] Сохранение summaries.txt
```

Для каждой статьи скрипт формирует промпт с заголовком и текстом, отправляет в модель и записывает ответ в выходной файл.

---

## Структура проекта

```
├── news_summarizer.py   # основной скрипт
├── bbc-news-data.csv    # входной датасет
├── summaries.txt        # результат (генерируется автоматически)
└── README.md
```

---

## Датасет

[BBC News Dataset](https://www.kaggle.com/datasets/pariza/bbc-news-summary) — 2225 новостных статей пяти категорий:

| Категория | Описание |
|-----------|----------|
| `business` | Бизнес и финансы |
| `entertainment` | Развлечения |
| `politics` | Политика |
| `sport` | Спорт |
| `tech` | Технологии |

Формат файла — TSV (разделитель `\t`), колонки: `category`, `filename`, `title`, `content`.

## Установка и запуск

### 1. Установка зависимости

```bash
pip install requests
```

### 2. Получение API-ключа Groq

На [console.groq.com](https://console.groq.com).


### 3. Запуск

```bash
python news_summarizer.py
```
